## Features

- Online Order
- Table Booking
- Responsiveness


## Run Locally

Clone the project

```bash
  git clone https://github.com/sohomofficial/little-lemon.git
```

Go to the project directory

```bash
  cd little-lemon
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```
